# Changelog

## [1.2.0] - 2025-04-22
- Add Monte Carlo pricer: European, Asian, and barrier options
- Add implied volatility solver (Brent's method via scipy)
- Add web UI for interactive pricing

## [1.1.0] - 2025-01-30
- Add all 5 Greeks: delta, gamma, theta, vega, rho
- Add put-call parity verification
- Add ATM vol surface placeholder

## [1.0.0] - 2024-08-15
- Initial release: Black-Scholes pricing for calls and puts
- Edge case handling for expired options (T=0)
